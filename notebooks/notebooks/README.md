# Notebooks

This folder contains the main notebooks of the credit risk modeling project.

## Notebook structure

1. `01_exploratory_data_analysis.ipynb`  
   Data understanding, missing values, target distribution, variable analysis and correlations.

2. `02_logistic_regression.ipynb`  
   Baseline interpretable model for Probability of Default prediction.

3. `03_custom_glmboost.ipynb`  
   Custom GLMBoost-inspired implementation for binary classification.

4. `04_tree_based_glmboost.ipynb`  
   Tree-based Gradient Boosting model used for performance comparison.

5. `05_model_interpretability.ipynb`  
   Feature importance, SHAP and LIME explanations.

## Confidentiality note

The original dataset used in the academic project comes from a professional banking environment and is not included in this repository.

The notebooks are designed to be reusable with an authorized public or synthetic credit risk dataset.
